# Jump High (aka Jumpy)

[logo]: img/icon.png **Jump High** is a simple **Python 3** Jumper-like game written with **Pygame**.

## Requirements
* [Python 3](https://www.python.org/downloads/)
```bash
python3 -m pip install -U pygame --user
```
